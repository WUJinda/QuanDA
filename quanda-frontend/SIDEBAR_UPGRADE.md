## 🎬 动画效果

### 1. Logo 呼吸动画
```scss
@keyframes pulse {
  0%, 100% { opacity: 1; }
  50% { opacity: 0.85; }
}
// 3s 循环
```

### 2. 背景渐变动画
```scss
@keyframes gradientShift {
  0%, 100% { opacity: 1; }
  50% { opacity: 0.8; }
}
// 15s 循环
```

### 3. 图标脉冲动画（激活状态）
```scss
@keyframes iconPulse {
  0%, 100% { transform: scale(1.1); }
  50% { transform: scale(1.15); }
}
// 2s 循环
```

### 4. 底部光泽动画
```scss
@keyframes shimmer {
  0% { background-position: -200% 0; }
  100% { background-position: 200% 0; }
}
// 3s 循环
```

### 5. Logo 淡入动画
```scss
@keyframes fadeInDown {
  from {
    opacity: 0;
    transform: translateY(-20px);
  }
  to {
    opacity: 1;
    transform: translateY(0);
  }
}
// 0.6s 一次
```

## 🎯 最佳实践

### ✅ 推荐做法

1. **保持动画流畅**
   - 使用 transform 和 opacity
   - 避免频繁触发重排

2. **颜色一致性**
   - 使用设计系统颜色
   - 保持渐变方向一致

3. **交互反馈**
   - 悬停有明显变化
   - 激活状态突出
   - 过渡自然流畅

### ❌ 避免做法

1. 不要修改核心渐变色
2. 不要移除动画效果
3. 不要改变圆角大小
4. 不要使用硬编码颜色
